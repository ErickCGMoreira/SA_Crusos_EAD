Tipos de dados

 

Nessa aula vamos mostrar quais são os tipos de dados. Quando programamos muitas vezes estamos manipulando dados. Vamos descobrir quais os dados que estamos manipulando, existe três tipos principais de dados, os números, caracteres e os booleanos. Os números são os algoritmos do dia-a-dia 1, 2 e 3, pode ser também números formados por múltiplos algarismo por exemplo, 80 que é um número formado por dois algorismos. Os caracteres A Z, dentro disso temos as strings. O diferencial das strings é que uma string é que ela contém vários caracteres, por exemplo no caractere temos somente a letra "A" e a letra "Z", já nas strings temos uma palavra formada, que são os conjunto das caracteres, por exemplo a palavra "Mãe", está palavra consiste três caracteres, ao inves desta palavra ser vista como três letras separadas, ela é vista como uma palavra inteira. Já os booleanos eles representa se os argumentos são verdadeiros ou falsos, por exemplo, está de dia no momento, sim está de dia, então é um argumento verdadeiro, ou por exemplo, está de noite no momento, não está, então é um argumento falso.
