Exemplo

 

Nessa aula gostaríamos de compartilhar um exemplo, nesse exemplo irá envolver tudo que aprendemos nas aulas anteriores, por exemplo, temos as variáveis, loop, condicional dentro do loop, operadores e os tipos de dados. Vamos executar o código, primeiro declaramos a variável, var = 0, voltando no exemplo da caixinha, onde temos uma caixinha e dentro dela tem o número zero, então a variável começa com o valor de zero, se "var > 2", a variável sendo zero, zero é menor que dois, então verdadeiro, nós vamos entrar nesse loop, dentro do loop iremos executar duas coisas, primeiro vamos escrever "X" e após escrever, vamos mudar o valor da variável, o valor dela agora será o valor atual dela mais um, então o valor novo será "1", como isso é um loop e não uma condicional, vamos voltar para "var > 2", atualmente essa variável tem o valor de "1", então "1" é menor que dois, então vamos executar novamente, iremos escrever outro "x" uma vez feito isso, vamos para próxima linha, "var=var+1", "1+1" é igual a dois, nossa variável tem um novo valor que é "2", como estamos dentro de um loop, vamos voltar para a primeira linha, se "var > 2" como a variável tem o mesmo valor, dois não é menor que dois, e sim igual, então não vamos mais executar o código, uma vez que o loop é falso, vamos sair e continuar na última linha, agora iremos escrever "Z". Esse é o final do código, o resultado é que escrevermos "X X Z" escrevemos "x" duas vezes porque essa foi a quantidade que o loop repetiu, e "Z" para finalizar nosso código.
